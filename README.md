## 📌 Project Overview

The **AI-Driven Virtual Assistant for Medical Center** is an intelligent chatbot system designed to enhance patient experience and streamline appointment management.

This system allows patients to:

* Book appointments
* Reschedule or cancel appointments
* Check doctor availability
* Get answers to common medical center queries

The assistant uses **AI with Retrieval-Augmented Generation (RAG)** to deliver accurate and context-aware responses.

## 🔍 How It Works

1. User sends a query via chatbot
2. Query is converted into embeddings
3. Similar data is retrieved from knowledge base
4. AI generates a contextual response
5. System performs actions (e.g., booking)
